# Player

播放mp4格式的视频文件（音视频同步）。

## Usage

编译后，进入Debug目录下运行cmd：

`player <filename>`

* `filename`：将需要播放的视频文件放到`player.exe`同一目录下，直接输入文件名；或使用文件完整路径。

* 若提示找不到dll文件，将相应dll文件放到`player.exe`同一目录下即可

## Dependency

* [FFMPEG](http://ffmpeg.org/)

* [SDL2.0](https://www.libsdl.org/download-2.0.php)

* [OpenAL](http://www.openal.org/)
